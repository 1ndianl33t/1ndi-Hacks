# URLProbe


*Take a list of URLs and returns their HTTP response codes*


## Install

With Go:

```bash
▶ go get -u github.com/1ndianl33t/1ndi-hacks/urlprobe
```
## usage
```bash
▶ cat urls.txt | urlprobe 
```
